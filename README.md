# Путешествие по Грузии • Июль 2026

Интерактивный таймлайн маршрута путешествия по Грузии с детальным расписанием и блоком для бронирования отелей.

## Маршрут

- **11 июля**: Тбилиси — прилёт
- **12-13 июля**: Степанцминда (Казбеги) — Гергетская Троица, Долина Джута
- **14-15 июля**: Кутаиси — каньоны Окаце и Мартвильский
- **16-17 июля**: Батуми — горная Аджария, концерт
- **18-19 июля**: Тбилиси — обратная дорога, вылет

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, адаптивная верстка)
- Google Fonts (Inter)
- Font Awesome 6 (иконки)

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы сохраняя структуру папок
3. В настройках репозитория включите GitHub Pages (ветка main, папка /root)
4. Сайт будет доступен по адресу: `https://nikita31415.github.io/valeron2026/`
